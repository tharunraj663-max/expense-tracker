# Personal Expense Tracker

A command-line Expense Tracker application built using Python, SQLite, and Pandas. This project helps users manage daily expenses, track spending patterns, and import expense records from Excel files.

## Features

* Add new expenses
* View all expenses
* Update existing expenses
* Delete expenses
* Calculate total expenses
* View category-wise expense summary
* Calculate expenses between selected dates
* Import expense records from Excel files
* Store data using SQLite database

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install required packages

```bash
pip install pandas openpyxl
```

3. Run the application

```bash
python main.py
```
